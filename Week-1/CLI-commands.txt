
1.	$ To create a hidden file or directory you use the commands but you add (.) before 
	the file or directory name. 

2.	$ attrib +h (filename): is to make a file or directory hidden.

3.	$ attrib -h (filename): is to make a file or directory visible. 

4.	$ mkdir –p: is to create multiple directories in one command line.

5.	$ printf + “text %s\n text” or echo –e “text”: is to append a message to a file .

6.	$ echo  -e + “text \n “: is to append a message to a file by adding the (\n) you go to a new line.
